Release Notes DPT V0R1

Release Notes for DPT Version 0.1

Release date: 17 Sep 2004

This version is incomplete but contains the the main 4GL components. The front end is fully functional. The purpose of releasing the system in this form is to illustrate the potential of the finished product, and get some user feedback.

This release contains only rudimentary documentation, but it does include a short demo script that you can follow through to see the key features in action for yourself.

Features:

  • Large range of language functionality
    • Procedures, include contexts, groups, subsystems
    • All flow control, expressions, most $functions
    • User IO, dummy strings, report generation, USE PROC
    • Lots more

  • Moderate range of system configuration features
    • Commands and control parameters
    • Messaging, audit trail
    • Multitasking, resource locking, thread types, socket control
    • Lots more

  • GUI front end
    • Editor
    • Step debugger
    • System navigator
    • Lots more
