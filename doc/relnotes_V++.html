DPT Future Release Plans

Last updated: April 2011

After a flurry of work through 2010-2011 leading up to V3.0, available developer time is again probably going to be quite limited, and it's back to just minor enhancements and fixes for the time being.

Next release contents
Nothing is anywhere near decided yet but the most likely candidates for the next big version are one or more of the following. In order of attractiveness to code, and therefore likeliness:

  • A sizeable chunk of the Janus SOAP User Language features
  • Socket mode for the Java API (i.e. connect to remote host)
  • SQL interface for all layers (including join engine UL hooks)

See also the wish list.
